---
title: Tags API
description: Public API endpoints for managing and retrieving post tags
---

The Tags API provides endpoints to list, retrieve, and query posts by tag in your Halo CMS.

## List Tags

Retrieves a paginated list of tags.

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8091/apis/api.content.halo.run/v1alpha1/tags?page=0&size=10" \
  -H "Accept: application/json"
```
</CodeGroup>

### Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number. Default is 0.
</ParamField>

<ParamField query="size" type="integer" default="0">
  Size number. Default is 0.
</ParamField>

<ParamField query="labelSelector" type="array">
  Label selector. e.g.: `hidden!=true`
</ParamField>

<ParamField query="fieldSelector" type="array">
  Field selector. e.g.: `metadata.name==halo`
</ParamField>

<ParamField query="sort" type="array">
  Sorting criteria in the format: `property,(asc|desc)`. Default sort order is ascending. Multiple sort criteria are supported.
</ParamField>

### Response

Returns a `TagVoList` object containing paginated tag results.

<ResponseField name="items" type="array">
  Array of tag objects.
  
  <ResponseField name="metadata" type="Metadata" required>
    Tag metadata including name, labels, and timestamps.
  </ResponseField>
  
  <ResponseField name="spec" type="TagSpec">
    Tag specification.
    
    <ResponseField name="displayName" type="string" required>
      Display name of the tag.
    </ResponseField>
    
    <ResponseField name="slug" type="string" required>
      Tag slug for URL.
    </ResponseField>
    
    <ResponseField name="color" type="string">
      Tag color in hex format (e.g., `#FF5733` or `#F57`).
    </ResponseField>
    
    <ResponseField name="cover" type="string">
      Cover image URL.
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="status" type="TagStatus">
    Tag status information.
    
    <ResponseField name="permalink" type="string">
      Permanent link to the tag.
    </ResponseField>
    
    <ResponseField name="postCount" type="integer">
      Total number of posts with this tag.
    </ResponseField>
    
    <ResponseField name="visiblePostCount" type="integer">
      Number of published and public posts with this tag.
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="postCount" type="integer">
    Number of posts with this tag.
  </ResponseField>
</ResponseField>

<ResponseField name="page" type="integer">
  Current page number, starts from 1.
</ResponseField>

<ResponseField name="size" type="integer">
  Size of each page.
</ResponseField>

<ResponseField name="total" type="integer">
  Total number of tags.
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages.
</ResponseField>

<ResponseField name="first" type="boolean">
  Whether this is the first page.
</ResponseField>

<ResponseField name="last" type="boolean">
  Whether this is the last page.
</ResponseField>

<ResponseField name="hasNext" type="boolean">
  Whether there is a next page.
</ResponseField>

<ResponseField name="hasPrevious" type="boolean">
  Whether there is a previous page.
</ResponseField>

## Get Tag by Name

Retrieves a single tag by its name.

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8091/apis/api.content.halo.run/v1alpha1/tags/{name}" \
  -H "Accept: application/json"
```
</CodeGroup>

### Path Parameters

<ParamField path="name" type="string" required>
  Tag name (unique identifier).
</ParamField>

### Response

Returns a `TagVo` object.

<ResponseField name="metadata" type="Metadata" required>
  Tag metadata.
</ResponseField>

<ResponseField name="spec" type="TagSpec">
  Tag specification.
</ResponseField>

<ResponseField name="status" type="TagStatus">
  Tag status.
</ResponseField>

<ResponseField name="postCount" type="integer">
  Number of posts with this tag.
</ResponseField>

## List Posts by Tag

Retrieves all posts with a specific tag.

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8091/apis/api.content.halo.run/v1alpha1/tags/{name}/posts?page=0&size=10" \
  -H "Accept: application/json"
```
</CodeGroup>

### Path Parameters

<ParamField path="name" type="string" required>
  Tag name (unique identifier).
</ParamField>

### Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number. Default is 0.
</ParamField>

<ParamField query="size" type="integer" default="0">
  Size number. Default is 0.
</ParamField>

<ParamField query="labelSelector" type="array">
  Label selector. e.g.: `hidden!=true`
</ParamField>

<ParamField query="fieldSelector" type="array">
  Field selector. e.g.: `metadata.name==halo`
</ParamField>

<ParamField query="sort" type="array">
  Sorting criteria in the format: `property,(asc|desc)`. Default sort order is ascending. Multiple sort criteria are supported.
</ParamField>

### Response

Returns a `ListedPostVo` object containing posts with the specified tag.
